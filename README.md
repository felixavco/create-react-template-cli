## React Template CLI 

### Instalation

    npm install -g react-template-cli
### Usage
To create a new project run: 

    rt init 

Then answer the questions, once the project is ready run 

    cd your-new-project
### Create a component
To create a component run:

    rt create component MyComponent
or 

    rt c c MyComponent

### Create a page
To create a component run:

    rt create page MyPage
or 

    rt c p MyComponent
